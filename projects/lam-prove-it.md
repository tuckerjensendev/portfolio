# LAM “Prove It”: 5-minute Demo (Docker Compose)

## What it is
A fast, runnable demo that walks through the LAM proof loop:
1) mint a key
2) ingest a document
3) request context with citations
4) decode citations and verify SHA-256 matches

## My role
- Built the demo flow and wrote the “do this, then expect that” README so anyone can feel the protocol quickly.
