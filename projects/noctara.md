# Noctara: Website + Desktop App

## What it is
Noctara includes:
- a Django + Vite/React website with a “deploy from README” flow
- a macOS desktop app (Electron + Vite/React) for stock tracking
- small backend-ish components (e.g., proxy/worker endpoints)
